Brython Sample (Python syntax to replace Javascript)

The button color changes using Jquery and Python Syntax instead of Javascript.

Frontend code used for changing the button CSS:

from browser import window, document

jq = window.jQuery

window.colors_index = 0

#Function that changes the color of the button
def change_button(event):
    colors = ["darkcyan", "green", "purple", "black"]
    next_color= colors[window.colors_index]
    if window.colors_index < len(colors) -1 :
        window.colors_index +=1
    else:
        window.colors_index = 0

    jq('#mybutton').css("background-color", next_color)


#Bind blick Button to the function
document['mybutton'].bind('click', change_button)
